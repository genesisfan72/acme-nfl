A Gutenberg plugin that contains a block that shows NFL teams, Conferences, and Divisions.

== Installation ==

1. Upload `acme-nfl` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the NFL List block to your page
